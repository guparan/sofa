Tutorials for SOFA

Description:

The aim of these Tutorials is to understand, step by step, what is SOFA and how it works.
Each Tutorial is a series of SOFA Scenes and present a specific feature. All these Tutorials have been categorized for your convenience.
Whithin each Tutorial, the Scenes are ordered from basic to advanced, adding more and more components.
How to:

First, choose a Category. If you're just starting with SOFA, we advise you to start with Step by Step.
At any moment you can change the Category with the top left selector. To come back to this page, press Home.
For each Scene, you can test it in runSofa or load it in the Modeler with the buttons above.
